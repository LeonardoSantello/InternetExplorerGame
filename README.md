# Sobre InternetExplorerGame
Internet Explorer é um projeto desenvolvido como trabalho de conclusão do curso de Ciência da Computação da UNIFEOB, turma de 2025.

O jogo traz uma estética inspirada na computação clássica, colocando o jogador no papel de um explorador digital que deve limpar a BIOS do computador, enfrentando vírus, corrupções e outras ameaças que assombram sistemas antigos.

# Instalação / Compilação

Este repositório contém todos os arquivos-fonte do jogo.
Para compilar, basta instalar o Godot 4.5, abrir o editor e importar o projeto diretamente a partir deste repositório. Assim, você poderá exportar para diversas plataformas — porém o jogo oferece suporte apenas para computadores, então escolha uma das seguintes opções:

- Windows

- Linux

- macOS

- Web

Configuração necessária antes de exportar

No painel de exportação do Godot, abra a seção “Recursos” (Resources) e adicione manualmente os arquivos TXT localizados em res://dsl/data.
Certifique-se de incluir os seguintes caminhos:

- res://dsl/data/cmos.txt

- res://dsl/data/virus.txt

- res://dsl/data/mal_cabeamento.txt

- res://dsl/data/cavalo_de_troia.txt

Instrução completa: res://dsl/data/cmos.txt, res://dsl/data/virus.txt, res://dsl/data/mal_cabeamento.txt, res://dsl/data/cavalo_de_troia.txt
